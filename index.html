Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


			puts "How many items do you want to see?"
			number = gets.to_i
			display_number = [*1..number]

			display_number.each do |foobar|
    
  			if foobar % 3 == 0 && foobar % 5 == 0
    			display_number[foobar - 1] = "Foobar"
			elsif foobar % 3 == 0
			display_number[foobar - 1] = "Foo"
			elsif foobar % 5 == 0
			display_number[foobar - 1] = "Bar"
			end
			end
  
			puts display_number
